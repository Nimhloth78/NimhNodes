# NimhNodes
A small suite of ComfyUI nodes I have tossed together for my own uses. I decided others might benefit from them as well.


## NanoGPT Node
This node allows you to use your NanoGPT api connection to use their text models within your workflow.

## Natural Language to Danbooru
This sends your crerated prompt through the same NanoGPT connection to convert your prompt into danbooru tags. Useful for switching from a model that understands natural language to one that is trained on tags.

## Save Folder String
Allows you to save files up to 5 folders deep.

## Save Text
Send your created prompt or text directly to be saved in a text file.
